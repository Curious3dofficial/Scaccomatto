ENGINES FOLDER

This folder is intended to store the chess engine executable used by the application.

The project uses the Stockfish chess engine for AI move generation.

Due to file size limitations, the Stockfish executable is not included in this portfolio.

To enable AI functionality:

1. Download Stockfish from:
   https://stockfishchess.org/download/

2. Extract the downloaded file.

3. Place the Stockfish executable file (stockfish.exe) inside this "engines" folder.

4. Run the Java application normally.

If the engine is not present, the application will still run, but AI opponent features may be disabled.
